## 왜 블로그를 시작했나

오래 전부터 엔지니어링 노트를 여기저기 분산해 두었는데, 한 곳에서
검색 가능하고 공개된 형태로 정리하고 싶었습니다. 그래서 GitHub Pages에
정적 블로그를 올립니다.

## 기술 스택

- **Next.js 15** (App Router, 정적 export)
- **MDX** — 글 안에 React 컴포넌트 삽입 가능
- **Tailwind CSS + Typography**
- GitHub Pages 배포

```ts
// 예시 코드
export function hello(name: string) {
  return `Hello, ${name}`;
}
```
